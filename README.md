# BudgetApp

Project parte do curso de programação em Python da freeCodeCamp. Este projeto é uma implementação simples de um gerenciador de gastos por categorias, feito em Python.
Ele permite registrar depósitos, saques, transferências entre categorias e gerar um gráfico de porcentagem de gastos em formato textual (ASCII).

O projeto foi desenvolvido com foco em prática de classes, métodos, listas, dicionários e formatação de strings.

# Funcionalidades

- Criar categorias de gastos
- Registrar depósitos e saques
- Transferir valores entre categorias
- Ver o saldo atual de cada categoria
- Exibir o histórico de transações formatado
- Gerar um gráfico de porcentagem de gastos por categoria

# Conceitos praticados

- Programação Orientada a Objetos (POO)
- Classes e métodos
- Listas e dicionários
- Laços for e while
- Manipulação de strings
- Formatação de saída no terminal
- Cálculos de porcentagem e arredondamento
